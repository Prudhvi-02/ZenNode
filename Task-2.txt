2. List top 3 soft-wares / tools that inspire you. Why do these tools inspire you?
JavaScript, GitHub, Python
i) JavaScript:
JavaScript inspires me because of its role as the backbone of web development. As a client-side scripting language, it enables the creation of dynamic and interactive user interfaces. The ability to run in web browsers makes JavaScript essential for modern web development. Its asynchronous capabilities, supported by frameworks like Node.js, facilitate server-side development as well. I appreciate the vibrant ecosystem of libraries and frameworks like React and Angular, which empower developers to build responsive and feature-rich applications.
ii) GitHub:
GitHub is a web-based platform for version control using Git. It has revolutionized collaborative software development by providing a centralized hub for code repositories, collaboration, and project management. GitHub's impact on the open-source community is immense, fostering collaboration among developers globally.
iii) Python: 
Python is a versatile, high-level programming language that has gained widespread adoption across various domains, including web development, data science, machine learning, and more. Its readability, simplicity, and extensive libraries make it a favorite among developers. Python's community-driven development model, which encourages collaboration and inclusivity, has played a crucial role in its success.
